# casein-md-study
Practical Molecular Dynamics (MD) simulation of Casein using GROMACS on Google Colab.

## 🚀 Quick Start (Google Colab)
Click the button below to open the setup notebook directly in Google Colab. This will guide you through the GROMACS installation and environment setup.

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/[YOUR_GITHUB_ID]/casein-md-study/blob/main/colab/01_setup_gromacs.ipynb)

## 📂 Project Structure
- `colab/`: Jupyter Notebooks for Google Colab.
- `scripts/`: Utility scripts (e.g., auto-backup to Google Drive).
- `inputs/`: GROMACS parameter files (.mdp) and topology files.
- `analysis/`: Analysis scripts for trajectory data (Coming in Vol. 3).
- `figures/`: Visualizations of simulation results.

## 🛠 Prerequisites
- A Google account to use Google Colab.
- Basic understanding of terminal commands (optional, as the notebooks guide you).
